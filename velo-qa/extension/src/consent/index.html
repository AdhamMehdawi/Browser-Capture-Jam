Privacy & Data Consent

Before using VeloCap, please review what data we capture and how it's handled.

What we capture when you record

  • Screen recording — video of your browser tab or screen
  • Console logs — errors, warnings, and debug output
  • Network requests — URLs, status codes, headers, and bodies
  • User actions — clicks, form inputs, and navigation

Sensitive data is automatically redacted before leaving your browser. Authorization headers, cookies, passwords, tokens, and API keys are replaced with [REDACTED].

Important

  • Data is only captured when you click Record — never in the background
  • Recordings are stored securely on Azure with AES-256 encryption at rest
  • You can export or delete all your data at any time from Settings
